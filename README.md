# Quant Finance Market Analyzer

A Python-based financial analysis tool that evaluates stock performance and portfolio risk using quantitative finance methods.

## Features

- automated stock market data retrieval
- annualized return calculations
- portfolio volatility analysis
- Sharpe ratio calculation
- correlation analysis between assets
- stock performance visualization

## Technologies

Python  
pandas  
numpy  
yfinance  
matplotlib  

## Quantitative Methods

This project applies concepts from Modern Portfolio Theory including:

- portfolio return estimation
- covariance-based risk modeling
- diversification analysis
- Sharpe ratio evaluation

## Assets Analyzed

Apple (AAPL)  
Nvidia (NVDA)  
Microsoft (MSFT)  
S&P 500 Index (SPY)
